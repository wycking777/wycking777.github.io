使用Hexo搭建个人博客

发表于2025-09-10|更新于2025-09-10|教程Hexo
|总字数:623|阅读时长:2分钟|浏览量:

🎯 前言

最新更新：2025年9月更新

经过不懈努力，我的个人博客终于成功上线！本文将完整分享从零开始搭建个人独立博客的全过程，内容精炼实用，请严格按照步骤操作。

🛠️ 环境准备

必备工具下载

  1. Node.js

    • 访问 官网下载
    • 选择LTS（长期支持版本）

  2. Git

    • 访问 官网下载
    • 安装时保持默认选项

环境验证

打开命令提示符（管理员模式），依次执行：

📕安装Hexo

回到桌面 右键寻找 Open Git Bash Here,执行:

🌐 GitHub仓库配置

  1. 创建仓库

  2. 登录 GitHub

  3. 点击 “New repository”

  4. 仓库命名：用户名.github.io (必须)

  5. 选择 Public

  6. 勾选 “Add a README file”

  7. 创建仓库

SSH密钥配置

  1. 生成SSH密钥

    • # 将your-email@example.com改成你的Github邮箱！
ssh-keygen -t rsa -C "your-email@example.com"

💻 本地博客开发

初始化项目

  1. 在任意位置新建文件夹，然后进入文件夹

  2. 右键空白处然后点Open Git bash here，输入

  3.  hexo init
 # 如果‘command not find’，就在前面加上npx，如：npx hexo init

  4. 访问 http://localhost:4000 查看效果。

🚀 部署到GitHub

配置部署信息

  1. 修改 _config.yml：

    •   deploy:
  type: git
  repository: git@github.com:用户名/用户名.github.io.git
  branch: main

    • 第一次可能需要配置账户:

    • git config --global user.email "邮箱"
git config --global user.name "用户名"

  2. 访问 https://用户名.github.io 查看线上博客。

⚙️ 基础配置

修改 _config.yml：

  •     # 站点信息
    title: 博客名称
    subtitle: 博客副标题
    description: 博客描述
    keywords: 关键词1,关键词2
    author: 作者名
    language: zh-CN
    timezone: Asia/Shanghai

    # URL设置
    url: https://用户名.github.io
    root: /
